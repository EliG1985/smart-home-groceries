# SmartHome Groceries Backend API

This is the Node.js/Express backend for SmartHome Groceries.

## Modules
- Inventory
- Shopping List
- Chat
- Reports
- Store

## Getting Started
- Install dependencies: `npm install`
- Start development: `npm run dev`

## Environment
- Uses Supabase for auth, storage, and realtime.
- Configure `.env` for API keys and endpoints.
