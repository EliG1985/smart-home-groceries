# SmartHome Groceries Mobile App

This is the Expo React Native mobile app for SmartHome Groceries.

## Modules
- Inventory
- Shopping List
- Chat
- Reports
- Store

## Getting Started
- Install dependencies: `npm install`
- Start development: `npx expo start`

## Environment
- Uses Supabase for auth, storage, and realtime.
- Configure `.env` for API keys and endpoints.
